⚙️ TVABDOU — Configuration

Les clés sont stockées localement dans ton navigateur (jamais envoyées ailleurs).

🟣 Twitch (obligatoire)

Crée une application sur la Console Développeur Twitch. Type : « Application Integration ». URL de redirection : http://localhost. Récupère le Client ID et génère un Client Secret.

🔴 YouTube (pour l'onglet Vidéos + notifs YouTube)

Active « YouTube Data API v3 » dans la Google Cloud Console, puis crée une clé API dans « Identifiants ».
